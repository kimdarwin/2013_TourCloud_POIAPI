이번편에서는 코원 D3 vs 갤럭시플레이어 vs 아이팟터치 4세대 의 비교를 해 보는 시간을 가지도록 하겠습니다. 앞 포스팅에서 코원 D3 에 대한 스펙이나 상세사항에 대하여 설명을 해 드렸는데요. 이번 포스팅에서는 현재 가장 이슈가 되는 MP3 플레이어들에 대한 이야기를 좀 해 볼까 합니다. 

최근 스마트폰의 열풍으로 인하여 모든 기능이 스마트폰에 다 들어간다고 생각하시는 분들이 많은데 스마트폰 같은 경우 1년이 지나면 배터리 수명에 관한 문제가 하나둘씩 생겨나면서 스마트폰의 배터리가 획기적으로 개선되지 않는한 멀티미디어 기능 측면에서는 다른 대안이 있어야 한다는 필요성에 의하여 애플, 삼성 등이 MP3 플레이어 시장에 적극적으로 뛰어들고 있는 실정입니다. 

그에 발맞추어 국내기업인 코원도 시장의 트랜드를 빨리 반영하여 MP3 플레이어 시장에 빅3로 급부상을 하고 있는 형편인데요. 세계적인 대기업과 국내 대기업의 양대 시장 산맥에서 국내 중견기업의 제품이 과연 살아남을 수 있을지 이 포스팅을 통하여 그 경쟁력을 알아보도록 하겠습니다.

먼저 각제품의 스펙부터 보시겠습니다

3가지 제품을 한눈에 비교하여 파악할 수 있는 일종의 도표입니다. ^^ 각 제품의 장담점을 열거해 보도록 하겠습니다. (지극히 개인적인 의견일 수 있습니다. 참고만 하시기 바랍니다. ^^)






아이팟터치 4세대

장점

1. 앱스토어를 사용하여 어플리케이션을 바로 다운받을 수 있으며 사용가능한 어플리케이션과 게임이 많음 (아이폰 사용하시는 분들에게는 유용할듯 )
2. 레티나 LCD를 사용하여 더욱 선명
3. Face타임을 이용하여 영상통화 가능 (애플사에서 가장 강력히 주장하는 기능)
4. 동영상 편집가능 (이 기능은 휴대용 기기에는 별로... ㅠ.ㅠ )
5. 감성적인 음악재생 (음악 재생방식이 다른기기와는 차별화되어 흔들면 다음곡으로 넘어감)
6. 터치감이 제일 좋음 (이거 아주 중요한 요소입니다. 개인적으로는 이 터치감이 제일 마음에 듭니다. )
7. 디자인이 가장 얇고 이쁨. (개인적인 생각)

단점

1. DMB기능을 지원하지않습니다. (DMB 많이 보는데 좀 아쉽죠 ㅠ.ㅠ )
2. 외장메모리 슬롯을 지원하지 않아서 메모리 확장이 불가능합니다 (정말 기술력이 떨어지지도 않을텐데.. 확장이 안 된다는거... ㅠ.ㅠ )
3. 동영상과 음악 재생을위해 인코딩이 필요합니다 (정말 짜증나는 일입니다. PSP 에 한동안 영화를 넣고 다녔는데.. 인코딩 이거 시간 잡아먹는게 장난 아니더군요.. ㅠ.ㅠ 인코딩 안하고 영화 안보고 맙니다. ㅠ.ㅠ )
4. 전자사전과 FM라디오 기능이 없습니다. 
5. 재생가능한 오디오확장자가 적습니다 (고음질의 음원파일 재생불가능 ex. FLAC/OGG/APE파일)
6. A/S비용이 비싸고 A/S받기 어렵습니다. (요즘 사후관리 안되면 좀 그렇죠?)
7. 액정이 3.5인치로 제일 작습니다. (비교제품중에서)






갤럭시 플레이어 (아직 정식발매되지는 않았습니다)

장점 

1. 액정이 4인치로 가장큽니다. (아주 중요한 요소죠 )
2. 장착되어있는 카메라가 가장 좋습니다. (카메라 요즘 많이 사용하는데.. 카메라 기능이 가장 좋아요 ^^)
3. 배터리가 착탈식이라 교체가 가능합니다. (갤럭시 플레이어가 재생 시간이 짧은 반면 이렇게 배터리를 갈아 끼울수 있는 시스템을 지니고 있습니다. )
4. GPS내장으로 네비게이션기능이 지원됩니다. (아주 획기적이죠.. ㅎㅎ 삼성은 뭘 잘 만든다니깐.. )
5. A/S를 쉽게 받을 수 있습니다. (삼성의 가장 큰 장점이죠 ^^)

단점

1. TV-OUT기능이 지원안됩니다. (아주 조금 아쉽습니다. )
2. 동영상 재생시간이 짧습니다. (이것도 정말 좀 그런데 그나마 배터리 교체로 위로를 받습니다. ㅠ.ㅠ )
3. 가격이 가장 비쌉니다. (뭐 이건 제품이 좋다면야.. )
4. 펌웨어 업데이트를 지속적으로 해주지 않습니다. (좀 빨리 해주면 좋으련만... ㅠ.ㅠ )
 






코원 D3

장점

1. 동영상과 음악재생에 있어서는 제일 좋습니다. (이건 코원 D3 의 가장 큰 장점입니다. 코원 D3 를 사게 되면 바로 이 기능이 훌륭하여 사게 되는거죠 )
2. Full HD파일(1080P)을 무인코딩으로 재생합니다. 또한 동영상 재생시간 10시간이라는 장점이 있습니다. (비슷한 맥락입니다. )
3. BBE+ / JetEffect3.0 음장으로 음질이 좋습니다. (음악 + 동영상의 우수함은 3제품 중 최고입니다. )
4. 동영상과 음악적 부분에서 생각한다면 가격대비 성능이 제일 좋습니다. 
5. 지속적인 펌웨어 업데이트로 사후관리가 철저합니다.
6. NTFS 포멧을 지원함으로써 4G이상의 파일을 넣을 수 있어서 고화질의 영상을 감상할 수 있습니다.
7. HDMI단자를 이용한 TV-OUT기능을 지원합니다. (이건 코원 D3 만 가능한 기능입니다. )
8. 아몰레드 LCD를 사용하여 밝습니다.
9. 동영상 재생시간이 깁니다.

단점

1. 터치감이 훌륭하지는 않습니다. (터치감 중요한데 터치감이 좀 안 좋더군요.. 이건 펌웨어 업글을 통하여 해결을 해 준다고 합니다. )
2. 카메라가 지원되지 않습니다. (카메라까지 지원되면 무조건 D3 구입하는건데... ㅠ.ㅠ )
3. Wi-Fi 802.11 n 이 지원되지 않아 인터넷이 조금 느린 감이 있씁니다. (Wi-Fi 802.11 n 지원 공유기의 경우에 한정)


이정도로 3가지 제품의 장단점을 들수 있겠습니다. 모든 제품에 장단점은 존재합니다. 아이폰을 사용하시는 분들은 아이팟 터치 4세대를 사용하시는 것이 효율적인 측면에서는 가장 좋겠습니다만 동영상 재생시 파일을 변환해야 하는 번거로움 (이거 안 해 보신 분들은 모릅니다. 디게 귀찮습니다. ㅠ.ㅠ ) 메모리 확장의 불가능, DMB 를 지원하지 않는 등의 눈에 보이는 단점이 있습니다. 하지만 디자인이 제일 이쁘고 터치감이 좋기 때문에 아이폰을 사용 하시는 분들은 아마 그래도 사용 하실 듯도 합니다. 

갤럭시 플레이어는 아직 출시가 되지 않았습니다만 많이 이들의 관심속에 가장 익숙한 플레어이가 되지 않을까 합니다. 가격이 조금 비싸다는 단점이 존재하고 동영상 재생 시간이 4시간으로 좀 긴영화 한편을 보면 배터리가 다 되는 단점이 존재합니다만 배터리를 갈수 있는 시스템을 도입하여 그 단점을 조금 극복할려는 노력이 돋보입니다. 

코원 D3 는 디자인이나 터치감 등에서는 위의 두 제품에 비하면 좀 딸리는 것은 사실입니다만 동영상과 음악 부분에서는 위의 2가지 MP3 플레이어에 비하여 가장 합리적인 대안이 될수 있지 않을까 하는 생각입니다. 하지만 코원 D3 는 동영상과 음악에 워낙 강점이 있다보니 카메라 같은 요즘 트랜드 반영에는 소극적인 듯 합니다. 

개인적으로 MP3 는 멀티미디어 기능 보다는 본 기기의 기능에 가장 충실해야 한다고 생각합니다. MP3 의 가장 충실한 기능은 음악일것입니다. 최근에는 음악과 더불어 동영상 재생까지 기본 기능으로 생각하는데요. 이런 기본 기능에 가장 충실한 제품이 코원 D3 가 아닐까 합니다. 

NTFS 포멧 방식을 지원하여 무인코딩으로 1080P 까지 재생이 가능하다는 것은 아주 합리적인 최근 트랜드를 가장 잘 반영한 듯한 느낌이 듭니다. 지하철을 타든 버스를 타던간에 이 기능은 가장 중요한 기능중에 기본적인 기능이 될테니까요. 

이렇게 코원 D3 vs 갤럭시플레이어 vs 아이팟터치 4세대 의 대표적인 3가지 MP3 플레이어에 대한 비교를 간단히 적어보았습니다. 이 포스팅은 MP3 플레이어를 구입하시는데 조금이나마 도움이 되고자 개인적인 생각을 많이 반영하여 작성하였습니다. 글을 보시고 참고는 하시되 제 생각이 절대적인 것은 아니니 잘 판단하여 좋은 MP3 플레이어를 구입하시기 바랍니다. ^^
